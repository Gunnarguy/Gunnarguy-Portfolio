# Website Fields

Field      Sub-Field      Required     Description
 Name       Name

 category                  No           Enum of predefined volunteering causes:
                                             PERSONAL
                                             COMPANY
                                             BLOG
                                             RSS
                                             PORTFOLIO
                                             OTHER


 label                     No           Localizable label a member chose for a website in "OTHER"
                                        category. It is a MultiLocaleString type.

 url                       Yes          Localized URLs for a website. It is a MultiLocaleUrl type.
