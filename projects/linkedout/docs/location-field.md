# Location Field

geoLocation Field (New)
Member's current location using Bing Geo taxonomy. This field is currently optional
while Bing Geo migration is in progress.


                                                                                    ﾉ    Expand table


 Field Name       Description                                                                Type

 geo              Member's current location defined by Bing Geo URN.                         Geo
                                                                                             URN

 autoGenerated    Default to true. True if existing geoLocation is back filled or dual       boolean
                  written from legacy location . False if it's a member input. This is
                  read-only, temporary and used for Bing Geo migration to indicate
                  whether the existing geo location is a member input. Clients may
                  suggest members to re-select their location when they haven't.

 Field Name                     Type
