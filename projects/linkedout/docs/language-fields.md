# Language Fields

Field Name     Required     Description

 id             Yes          The unique identifier of the language object.

 name           Yes          Localizable language name. It is a MultiLocaleString type.

 proficiency    No           Member's proficiency level of the language. Possible enum values:
                                 ELEMENTARY (Elementary proficiency)
                                 LIMITED_WORKING (Limited working proficiency)
                                 PROFESSIONAL_WORKING (Professional working proficiency)
                                 FULL_PROFESSIONAL (Full professional proficiency)
                                 NATIVE_OR_BILINGUAL (Native or bilingual proficiency)
