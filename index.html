The Open- Suite

OpenResponses

The flagship native iOS client for the OpenAI Responses API. A powerful playground for developers and power users to explore the latest AI capabilities.

  • Multi-Model Support: Seamless switching between GPT-4o, o1-preview, and Computer Use models.
  • Advanced Tooling: Features "Computer Use" for browser automation, Code Interpreter, and Web Search.
  • MCP Integration: First-class support for Model Context Protocol to connect with Notion, GitHub, and more.
  • Real-Time Streaming: Granular status updates, token estimation, and interactive activity feeds.
SwiftUI OpenAI Responses API MCP Computer Use WebSockets
GitHub

OpenIntelligence

A privacy-first RAG system for iOS 18 that orchestrates on-device intelligence with hybrid search.

  • Hybrid Search Engine: Combines BM25 (keyword) and Vector Search with Reciprocal Rank Fusion (RRF).
  • Privacy by Design: All processing happens on-device or via Private Cloud Compute (PCC).
  • Multi-Backend: Supports Apple Intelligence, Local GGUF (Llama 3), and OpenAI.
  • Agentic Tooling: 12+ tools allowing the LLM to search, summarize, and analyze your library.
Swift 6 Actors Core ML Local LLMs RAG
GitHub

OpenCone

On-device RAG that turns personal documents into a searchable knowledge base using Pinecone and OpenAI.

  • Multi-Format Ingestion: Processes PDF, DOCX, Images (OCR), and Code with semantic chunking.
  • Vector Search: High-performance retrieval using Pinecone vector database.
  • Observability: Structured logging and real-time processing stats for full transparency.
  • Secure Storage: Security-scoped bookmarks and Keychain integration for credentials.
SwiftUI Pinecone VisionKit PDFKit OpenAI
App Store GitHub

OpenAssistant

A feature-rich native client for the OpenAI Assistants API, managing the full lifecycle of AI assistants.

  • Assistant Management: Create, configure, and manage assistants with specific instructions and models.
  • Vector Stores: Full CRUD operations for vector stores and file associations.
  • Code Interpreter: Enable advanced data analysis and code execution capabilities.
  • Robust Architecture: Built with MVVM and Combine for a reactive, reliable experience.
